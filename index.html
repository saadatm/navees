Install

Windows
  1. Download the file navees-windows-<version>.zip from the latest release page.
  2. Unzip the file, and run navees-installer-<version>.exe. Make sure that Add to language bar list is checked, and then click Install. The system will display a confirmation message after successful installation.
  3. A little button saying EN (or ENG) will appear on the taskbar—near the system tray/status area. (You may need to restart your computer before it appears.) Click it, and select اُر (or UR or URD).
  4. Type!

Linux

Navees is currently included in the following Linux distributions:

  • Arch Linux
  • Fedora 34+
  • Manjaro
  • openSUSE Tumbleweed
  • Ubuntu 22.04+

In order to enable Navees, follow the instructions for your desktop environment below.

GNOME 40

  1. Open the Activities overview and start typing Terminal.
  2. Click on Terminal to open a terminal window, and run the following command: 
    gsettings set org.gnome.desktop.input-sources show-all-sources true
    (This is needed so that GNOME can load the “extra” keyboard layouts, in which Navees is included.)
  3. Open the Activities overview again and start typing Settings.
  4. Click on Settings.
  5. Click Keyboard in the sidebar.
  6. In the Input Sources section, click the + button.
  7. Click the ⋮ button and search for Urdu. Then select either Urdu (Pakistan) or Urdu (India).
  8. Select Urdu (Navees), and click the Add button.
  9. A layout indicator will appear in the top bar. Click it, and select Urdu (Navees).
  10. Type!

KDE Plasma 5

  1. Go to System Settings > Input Devices > Keyboard, and click the Layouts tab.
  2. Enable Configure layouts (if it is not already checked), then click the Add button.
  3. In the Limit selection by language drop-down list, select Urdu.
  4. In the Layout drop-down list, select either Urdu (Pakistan) or Indian.
  5. In the Variant drop-down list, select either Urdu (Navees, Pakistan) or Urdu (Navees).
  6. Click OK, and then Apply.
  7. A keyboard layout indicator will appear in the System Tray. Click it until it shows either pk or in.
  8. Type!

Android

Navees (adapted for mobile) is available in the following Android keyboards:

  • Simple Keyboard (F-Droid, Google Play)
  • Indic Keyboard (F-Droid, Google Play)

Documentation

Navees is based on the CRULP Phonetic Keyboard v1.1 with the following changes:

  • Added ؤ (U+0624 ARABIC LETTER WAW WITH HAMZA ABOVE) at AltGr + W
  • Added ۂ (U+06C2 ARABIC LETTER HEH GOAL WITH HAMZA ABOVE) at AltGr + O
  • Added ۓ (U+06D3 ARABIC LETTER YEH BARREE WITH HAMZA ABOVE) at AltGr + Y
  • Added أ (U+0623 ARABIC LETTER ALEF WITH HAMZA ABOVE) at AltGr + A
  • Added ؍ (U+060D ARABIC DATE SEPARATOR) at Shift + ,
  • Added ZWNJ (U+200C ZERO WIDTH NON-JOINER) at Shift + Space
  • Added ZWJ (U+200D ZERO WIDTH JOINER) at AltGr + |
  • Added RLM (U+200F RIGHT-TO-LEFT MARK) at AltGr + }
  • Added LRM (U+200E LEFT-TO-RIGHT MARK) at AltGr + {
  • Added ` (U+0060 GRAVE ACCENT) at `
  • Added ~ (U+007E TILDE) at AltGr + ~
  • Added $ (U+0024 DOLLAR SIGN) at AltGr + $
  • Added ^ (U+005E CIRCUMFLEX ACCENT) at AltGr + ^
  • Moved ؂ (U+0602 ARABIC FOOTNOTE MARKER) from AltGr + W to AltGr + F
  • Moved ؁ (U+0601 ARABIC SIGN SANAH) from AltGr + Y to AltGr + K
  • Removed the ligature ﷲ (U+FDF2 ARABIC LIGATURE ALLAH ISOLATED FORM) from AltGr + A
